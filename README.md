# Customer Service Scripts Website

A professional website for managing customer service workflows and scripts.

## Features

✅ **5 Essential Steps:**
- Step 1: Greeting
- Step 2: Verify Customer
- Step 3: Listen & Understand
- Step 4: Provide Solution
- Step 5: Closing

✅ **Rules for Each Step** - Display best practices and compliance requirements

✅ **One-Click Copy** - Click any script button to copy it to your clipboard

✅ **Professional Design** - Clean, modern, and mobile-responsive

✅ **Visual Feedback** - Shows "Copied!" confirmation when script is copied

## How to Use

1. Open `index.html` in your web browser
2. Click on any "Copy Script" button to copy that step's script
3. Paste it into your customer service chat or communication tool
4. Follow the rules displayed for each step

## Customization

You can easily customize:

### Edit Scripts
Open `index.html` and find the `script-text` class. Modify the text inside the quotes.

### Add More Steps
Copy and paste a `step-card` div and update:
- `step-number`
- `h2` title
- Rules in the `rules` section
- Script text in `script-text`

### Change Colors
Edit the colors in `style.css`:
- Primary color: `#667eea` (blue-purple)
- Secondary color: `#764ba2` (purple)

## Deploy to GitHub Pages

1. Go to your repository Settings
2. Scroll to "GitHub Pages"
3. Select "Deploy from a branch"
4. Choose "main" branch
5. Your site will be live at: `https://yazanhammami7-alt.github.io/customer-service-scripts/`
